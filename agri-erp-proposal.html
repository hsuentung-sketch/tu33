功能模組詳細說明
🔧
維修管理
工單建立、技師派工、工時記錄、零件領用、完工確認。支援 LINE 語音/文字快速報工，替代現有手寫流程。
工單 派工 工時 零件領用 維修紀錄
🛒
整機銷售
新機與二手機分開管理。品項包含割草機、鏈鋸等。報價單支援客戶等級自動調整售價，建立銷貨單後觸發庫存扣減與發票開立。
新機 二手機 報價單 銷貨單
📦
零件庫存管理
初期優先管理高單價 / 高周轉零件。進貨、銷售、維修領用自動扣庫存。安全庫存預警，支援定期盤點作業。
庫存異動 安全庫存 盤點 進貨管理
👥
客戶主檔與等級
三級定價體系：專業維修客戶、常客、散戶。每筆交易自動記錄，等級依消費頻率或人工調整。歷史維修紀錄隨時查閱。
專業維修 常客 散戶 交易歷史
💰
帳款管理
應收應付月結，自動計算到期日。到期前預警通知（LINE 推播）。支援分期付款與分批收款記錄。
月結 到期預警 收款記錄
🧾
電子發票
B2B 三聯式發票（統一編號）與 B2C 二聯式發票（手機載具 / 自然人憑證）。維修結案、銷貨建單後自動觸發開立流程。
B2B 三聯 B2C 二聯 手機載具 自動開立
🤖
AI 語意解析
技師用 LINE 說或打：「幫客戶阿明換了 6336 油封，工時一小時，另加機油兩罐」→ AI 自動解析並填入報工單，確認後送出。
語音輸入 文字解析 自動填單
📊
報表與儀表板
月業績概覽、維修收入分析、庫存周轉率、應收款齡分析。後台網頁與 LINE 推播每日/週報摘要。
業績報表 庫存分析 每日摘要

整機銷售管理
🆕
新機銷售
品項主檔維護（割草機、鏈鋸、耕耘機等），建議售價 + 成本分開管理。 報價單依客戶等級自動計算售價，轉銷貨單後扣整機庫存，觸發應收與發票。
品項主檔 自動報價 庫存扣減 發票開立
🔄
二手機銷售
二手機獨立建檔（機型、序號、收購成本、整備費用、外觀成色評級）。 可建立單一品項的報價與銷售，整備維修費用自動計入成本。
序號管理 整備成本 成色評級
📋
報價單流程
LIFF 表單或 LINE 語意輸入均可建立報價單。客戶確認後一鍵轉銷貨單，有效期可設定（預設 30 天），到期未轉自動標記失效。
LIFF 表單 LINE 語音 轉銷貨單 有效期管控
機型分類 管理方式 庫存單位 備註
割草機（新）型號主檔 + 數量台可含配件組合
鏈鋸（新）型號主檔 + 數量台
耕耘機（新）型號主檔 + 數量台
割草機（二手）單機序號建檔台（唯一）可記錄收購來源
其他二手機單機序號建檔台（唯一）整備工單關聯
零件庫存管理策略
初期不求全面，以高周轉 + 高單價品項優先，逐步擴展
📌 初期導入策略：ABC 分類優先 A 類（高單價 / 高周轉）：化油器、點火模組、油封組、刀盤、引擎等 → 先全部納管
B 類（中等）：濾芯、皮帶、螺絲組 → 第二階段納管
C 類（低值耗材）：機油、黃油、螺絲 → 可簡化管理或不納管
📥
進貨管理
LINE 掃碼或手動建立進貨單，多家供應商，自動更新庫存與成本均價。
🔍
安全庫存預警
每件零件設安全庫存量，低於閾值自動 LINE 推播提醒採購，附上建議採購量。
📋
盤點作業
支援定期盤點（按分類），LINE 掃碼或手動輸入實際數量，系統自動計算盤盈虧。
🔧
維修領用扣庫
技師在維修工單中選用零件，確認後自動從庫存扣減，維修成本同步計算。
📊
庫存分析
周轉率、滯銷品報表，協助優化採購策略，避免資金積壓在呆料。
🔗
品牌 / 型號對應
零件與適用機型建立對應關係，維修時自動推薦相容零件，加速查詢。

✅ 客戶確認清單
請與客戶逐項確認後，再進入 Claude Code 開發階段
  • 客戶等級名稱：「專業維修、常客、散戶」三級是否符合？還是有不同稱謂？
  • 工時費標準：各等級工時費價格區間是多少？
  • 二手機整備流程：收購後是否有固定整備工序需要記錄？
  • 零件料號格式：目前使用的料號規則是什麼？（如品牌縮寫 + 編號）
  • A 類零件清單：初期要納管哪些高單價 / 高周轉零件？約幾項？
  • 供應商數量：目前主要零件供應商有幾家？
  • 電子發票：B2C 要支援手機載具嗎？B2B 需開三聯式嗎？
  • 到府維修：有到府維修服務嗎？需要記錄地點與行程？
  • 多技師：店內有幾位技師？需要技師業績個別統計嗎？
  • 機器序號管理：新機需要登記序號 / 保固日期嗎？
  • LINE 帳號：是否已有 LINE 官方帳號？或需要新申請？
  • 現有 Excel：現有 Excel 資料（客戶 / 零件）是否需要轉移匯入？
