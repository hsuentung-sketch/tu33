# 部署指引 — 2026-04-17 版

本次新增功能：
- **短連結**：所有 PDF 下載連結改以 `https://erp-line-bot.onrender.com/s/<code>` 顯示
- **產品文件**：每個產品可上傳 PDS / SDS / DM / OTHER（PDF 或圖片，10MB 上限）
- **操作紀錄 / 異常紀錄**後台頁（ADMIN 限定）

---

## 1. Supabase SQL Editor — 建表

打開 https://app.supabase.com → 專案 → **SQL Editor** → New query，複製貼上：

```
docs/migrations/2026-04-17-shortlink-product-docs-error-log.sql
```

Run 之後執行：

```sql
SELECT tablename FROM pg_tables
WHERE tablename IN ('ShortLink','ProductDocument','ErrorLog');
```

應該列出三張表。

## 2. Supabase Storage — 建立產品文件 bucket

Supabase → **Storage** → New bucket：

| 欄位 | 值 |
|------|----|
| Name | `product-docs` |
| Public bucket | **否**（關閉，走 signed URL） |
| File size limit | 10 MB |
| Allowed mime types | `application/pdf, image/*` |

> 後端用 service-role key 上傳／簽章，不開 public，LINE 下載透過短連結 → signed URL（7 天）。

## 3. Render 環境變數

到 Render dashboard → `erp-line-bot` → Environment，新增：

| Key | 來源 | 備註 |
|-----|------|------|
| `SUPABASE_URL` | Supabase → Settings → API → Project URL | 形如 `https://xxxx.supabase.co` |
| `SUPABASE_SERVICE_ROLE_KEY` | 同頁的 `service_role` key | **不是** anon key；Service role 可繞過 RLS |
| `SUPABASE_PRODUCT_DOCS_BUCKET` | `product-docs` | 可留空，程式預設此值 |

其他 env 不變（`DATABASE_URL`、`JWT_SECRET`、LINE channel 設定維持）。

## 4. 部署

```bash
git push origin main
```

Render 會自動 build（已改為 `npm ci --include=dev && npx prisma generate && npm run fonts:download && npm run build`）。

Build log 有 `✔ Generated Prisma Client` 後可進入 service。

## 5. 驗收檢查

- [ ] `/admin` 登入後，側欄多出「操作紀錄」、「異常紀錄」兩項（ADMIN）
- [ ] 產品頁每列右側出現「文件」按鈕，可上傳 PDF 並看到列表
- [ ] LINE 輸入「查詢產品」→「全部」 → 應回 Flex 卡片（而非純文字），卡片下方有下載按鈕（若該產品已上傳文件）
- [ ] 銷貨單／進貨單／報價單送出後，PDF 連結為 `/s/<code>` 格式
- [ ] 點短連結 → 302 轉到真正 PDF → 中文字體正常
- [ ] 故意製造一筆 500 錯誤（例：stop Supabase 一下再恢復）→ 後台異常紀錄可看到

## 6. 回滾方案

SQL 全部用 `CREATE TABLE IF NOT EXISTS` 與 `DO $$ … $$` 保護，可重跑不會炸。
若需完全撤除：

```sql
DROP TABLE IF EXISTS "ErrorLog";
DROP TABLE IF EXISTS "ProductDocument";
DROP TYPE IF EXISTS "DocumentType";
DROP TABLE IF EXISTS "ShortLink";
```

程式層面：`git revert` 本次 commit 即可，新程式相容舊 DB（table 不在 → Prisma 會拋錯，短連結／文件功能 unavailable 而已，既有銷貨／進貨／帳務不受影響）。
